# Project 1 — Capture the Flag Write-Up

**Course:** CMIT 321 Ethical Hacking  
**Due:** Week 3 (individual)  
**Status:** _in progress_

## Assignment summary

Identify one or two CTF challenges you didn't attempt or couldn't complete. Document the techniques, tools, websites, or other resources that can be used to solve them. Correctly answer 10 questions from the in-house CTF quiz bank and submit the company survey.

## Deliverables

- [ ] Selected 1–2 unfinished or unattempted CTF challenges
- [ ] Written walkthrough for each selected challenge (see `challenges/`)
- [ ] List of tools, techniques, and references used (`tools-and-resources.md`)
- [ ] Personal reflection on the 10 quiz-bank questions (`ctf-quiz-reflection.md`)
- [ ] Company survey submitted in the UMGC classroom (tracked here, not stored)

## Folder contents

| File / Folder | Purpose |
| --- | --- |
| `challenges/challenge-01.md` | Walkthrough template for the first selected challenge |
| `challenges/challenge-02.md` | Walkthrough template for the second selected challenge |
| `tools-and-resources.md` | Tooling and reference list used across both challenges |
| `ctf-quiz-reflection.md` | Reflection on the 10 quiz-bank topics (no answer keys) |
| `assets/` | Screenshots and diagrams (redacted) |

## Notes on integrity

- No flag values, quiz answer keys, or copyrighted UMGC content are committed.
- The point of this folder is to demonstrate **method** — how the challenge would be solved — not to publish solutions to live CTF infrastructure.
